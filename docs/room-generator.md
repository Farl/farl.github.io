# Maze Runner

A multiplayer 3D maze runner with procedurally generated rooms.

- **Demo**: https://farl.github.io/room_generator/
- **Source Code**: https://github.com/Farl/room_generator

![Maze Runner](/img/portfolio/room-generator.png)

## About

Each session generates a fresh maze of rooms. Run through it with other players, find the exit, and try not to get lost.
