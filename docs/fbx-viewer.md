# FBX Viewer

An in-browser FBX model viewer with animation playback.

- **Demo**: https://farl.github.io/fbx_viewer/
- **Source Code**: https://github.com/Farl/fbx_viewer

![FBX Viewer](/img/portfolio/fbx-viewer.png)

## About

Drag-and-drop an .fbx file to inspect 3D models in the browser. Inspect the rig, switch between embedded animations, and orbit around the model.
