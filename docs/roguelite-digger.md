# 挖地瓜 Roguelite

A digging-themed roguelite where you keep harvesting to push deeper.

- **Demo**: https://farl.github.io/roguelite-digger/
- **Source Code**: https://github.com/Farl/roguelite-digger

![挖地瓜 Roguelite](/img/portfolio/roguelite-digger.png)

## About

向下挖掘、收集地瓜、升級裝備，每次失敗都會留下一些進度。輕量的 roguelite 玩法，用來實驗節奏與升級曲線。
