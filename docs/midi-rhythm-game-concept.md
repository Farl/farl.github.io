# MIDI Rhythm Game Studio

A concept rhythm game that turns any MIDI file into playable charts.

- **Demo**: https://farl.github.io/midi_rhythm_game_concept/
- **Source Code**: https://github.com/Farl/midi_rhythm_game_concept

![MIDI Rhythm Game Studio](/img/portfolio/midi-rhythm-game-concept.png)

## About

Load a MIDI file, watch it become a playable rhythm chart automatically, and experiment with how different note streams map to gameplay.
