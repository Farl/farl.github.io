# Multiplayer Snake 3D

The classic snake game reimagined as 3D multiplayer.

- **Demo**: https://farl.github.io/multiplayer_snake_3d/
- **Source Code**: https://github.com/Farl/multiplayer_snake_3d

![Multiplayer Snake 3D](/img/portfolio/multiplayer-snake-3d.png)

## About

Snake, but in 3D and with other players in the same arena. Grow longer, dodge other snakes, and survive.
