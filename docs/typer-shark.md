# Typer Shark

An AI-generated word-list typing game in the spirit of Typer Shark.

- **Demo**: https://farl.github.io/typer-shark/
- **Source Code**: https://github.com/Farl/typer-shark

![Typer Shark](/img/portfolio/typer-shark.png)

## About

Defeat advancing sea creatures by typing words. The word lists are generated by AI, so the vocabulary changes each session and can be themed on demand.
