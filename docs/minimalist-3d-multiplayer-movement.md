# Minimalist 3D Multiplayer Movement

A bare-bones experiment in real-time 3D multiplayer movement.

- **Demo**: https://farl.github.io/minimalist_3d_multiplayer_movement/
- **Source Code**: https://github.com/Farl/minimalist_3d_multiplayer_movement

![Minimalist 3D Multiplayer Movement](/img/portfolio/minimalist-3d-multiplayer-movement.png)

## About

Just movement and presence — a stripped-down 3D space where players can join, walk around, and see each other in real time. Useful as a starting point for larger multiplayer worlds.
