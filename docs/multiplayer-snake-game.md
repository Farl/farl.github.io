# Multiplayer Snake

A classic 2D multiplayer snake battle in the browser.

- **Demo**: https://farl.github.io/multiplayer_snake_game/
- **Source Code**: https://github.com/Farl/multiplayer_snake_game

![Multiplayer Snake](/img/portfolio/multiplayer-snake-game.png)

## About

Multiple players, one shared arena, and the classic snake mechanic. Each player tries to outlast the others without crashing into walls or tails.
