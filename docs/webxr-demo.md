# WebXR Demo

A lightweight WebXR scene that runs in headsets and browsers.

- **Demo**: https://farl.github.io/webxr_demo/
- **Source Code**: https://github.com/Farl/webxr_demo

![WebXR Demo](/img/portfolio/webxr-demo.png)

## About

A small WebXR sample built to run on standalone VR headsets and standard browsers. Useful as a baseline for testing WebXR support in different environments.
