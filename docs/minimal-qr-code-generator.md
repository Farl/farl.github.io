# Advanced QR Code Generator

A minimal-yet-flexible QR code generator with logo overlay and styling options.

- **Demo**: https://farl.github.io/minimal-qr-code-generator/
- **Source Code**: https://github.com/Farl/minimal-qr-code-generator

![Advanced QR Code Generator](/img/portfolio/minimal-qr-code-generator.png)

## About

Generate QR codes with custom colors, error correction levels, and an optional centered logo. The interface stays minimal but exposes the controls you need for branded codes.
