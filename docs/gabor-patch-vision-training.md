# Gabor Patch Vision Training

A visual training tool using Gabor patches to challenge contrast sensitivity.

- **Demo**: https://farl.github.io/gabor_patch_vision_training/
- **Source Code**: https://github.com/Farl/gabor_patch_vision_training

![Gabor Patch Vision Training](/img/portfolio/gabor-patch-vision-training.png)

## About

Adjustable Gabor patches at varying contrasts and orientations, intended for self-directed visual acuity and contrast-sensitivity training.
