# Imgly Background Removal Demo

Client-side background removal powered by img.ly's WASM model — no server required.

- **Demo**: https://farl.github.io/imgly-background-removal-demo/
- **Source Code**: https://github.com/Farl/imgly-background-removal-demo

![Imgly Background Removal Demo](/img/portfolio/imgly-background-removal-demo.png)

## About

Drop in any image and remove its background entirely in the browser using img.ly's WebAssembly background-removal model. Nothing is uploaded to a server.
