# 3D Audio Particle Visualizer

A 3D particle system that dances to live audio input.

- **Demo**: https://farl.github.io/audio_particles/
- **Source Code**: https://github.com/Farl/audio_particles

![3D Audio Particle Visualizer](/img/portfolio/audio-particles.png)

## About

Press play and watch thousands of particles react in real time to the audio's frequency spectrum. Built around the Web Audio API and a GPU-accelerated particle system.
