# Morph Anything

Smoothly morph between any two images with AI-driven keyframes.

- **Demo**: https://farl.github.io/morph_anything/
- **Source Code**: https://github.com/Farl/morph_anything

![Morph Anything](/img/portfolio/morph-anything.png)

## About

Pick a start and end image and let AI generate the morph between them. The result is exported as a smooth animation.
