# 2D Groove Party

A browser-based 16x16 music grid with live multiplayer cursors and shared tile triggers.

- **Demo**: https://farl.github.io/2d_groove_party___disco_multiplayer/
- **Source Code**: https://github.com/Farl/2d_groove_party___disco_multiplayer

![2D Groove Party](/img/portfolio/2d-groove-party.png)

## About

Tap tiles on a shared 16x16 grid to layer beats, melodies, and effects together with everyone else in the room. Each player sees the others' cursors live, so jam sessions feel like a tiny disco.
