# AI Visual Novel Generator

Generate a complete visual novel — story, characters, and backgrounds — from a single theme.

- **Demo**: https://farl.github.io/ai_visual_novel_generator/
- **Source Code**: https://github.com/Farl/ai_visual_novel_generator

![AI Visual Novel Generator](/img/portfolio/ai-visual-novel-generator.png)

## About

Type in a theme and the AI writes a full story, generates character art, and creates background scenes for a playable visual novel. A two-step generation pipeline takes the story from outline to full script.
