# Katamari Roll

Roll up everything in a tiny Katamari-style sandbox.

- **Demo**: https://farl.github.io/basic_katamari/
- **Source Code**: https://github.com/Farl/basic_katamari

![Katamari Roll](/img/portfolio/basic-katamari.png)

## About

A miniature Katamari prototype: roll a sticky ball around a small 3D scene, picking up smaller objects until you grow large enough to roll up the bigger ones.
