# Pose Editor

Edit and pose 3D character rigs interactively.

- **Demo**: https://farl.github.io/pose_editor/
- **Source Code**: https://github.com/Farl/pose_editor

![Pose Editor](/img/portfolio/pose-editor.png)

## About

Manipulate joint rotations on a 3D character rig directly in the browser. Useful for quickly building reference poses or animation keyframes.
