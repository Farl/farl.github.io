# Image Sliding Puzzle

Classic sliding-tile puzzle generated from any image you choose.

- **Demo**: https://farl.github.io/image-sliding-puzzle/
- **Source Code**: https://github.com/Farl/image-sliding-puzzle

![Image Sliding Puzzle](/img/portfolio/image-sliding-puzzle.png)

## About

Upload or pick an image and the page turns it into a sliding puzzle. Choose grid size, shuffle, and slide tiles back into place.
