# Pose Maker

FBX-based pose authoring tool.

- **Demo**: https://farl.github.io/pose_maker/
- **Source Code**: https://github.com/Farl/pose_maker

![Pose Maker](/img/portfolio/pose-maker.png)

## About

Load FBX rigs and author poses with a streamlined timeline and joint controls. Companion tool to the Pose Editor for producing reusable poses.
