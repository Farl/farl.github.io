# AI Hidden Object Game

A hidden-object game where AI vision detects objects in any image you upload or generate.

- **Demo**: https://farl.github.io/ai_hidden_object_game/
- **Source Code**: https://github.com/Farl/ai_hidden_object_game

![AI Hidden Object Game](/img/portfolio/ai-hidden-object-game.png)

## About

Upload your own image — or generate one with AI — and let an AI vision model find the hidden objects and bounding boxes. The result is a fresh hidden-object scene every time.
