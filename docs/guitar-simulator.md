# Guitar Simulator

Play a virtual guitar in the browser with chord and strum controls.

- **Demo**: https://farl.github.io/guitar_simulator/
- **Source Code**: https://github.com/Farl/guitar_simulator

![Guitar Simulator](/img/portfolio/guitar-simulator.png)

## About

A web-based guitar with chord shapes, strum gestures, and live playback. Useful for noodling without a guitar nearby.
