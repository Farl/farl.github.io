# RealTime MotionEraser

Mobile-optimized real-time video motion eraser.

- **Demo**: https://farl.github.io/motioneraser---keep-frame-video-processor/
- **Source Code**: https://github.com/Farl/motioneraser---keep-frame-video-processor

![RealTime MotionEraser](/img/portfolio/motioneraser.png)

## About

A "keep-frame" video processor that suppresses motion in a video stream — keeping the static parts of the scene while erasing whatever moves through it. Tuned for performance on mobile.
